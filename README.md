##Alloy-Eyes介绍

Alloy Eyes是一款基于T210核心板（主控芯片为K210）用于人脸检测识别、物体识别的快速验证的开发板;同时内置高精度的人脸检测识别算法，同时支持Arduino平台，让开发更简单。
